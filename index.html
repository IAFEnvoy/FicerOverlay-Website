Core Features

The software has a plenty of useful features.

Highly Optimized

Use Tauri, one of the new frameworks based on Webview, to be faster in speed and smaller in resource occupancy.

Powerful Cache System

All data fetched from APIs will be cached and have a reasonable expire time.

More Game Mode

Support Bedwars, Sky War, Murder Mystery, Duel, UHC, Mega Wall, Wool War, Blitz Survival Game and all their sub-modes.

Stats Sort

You can sort any stat by clicking the header of the table.

All Stats Searcher

Open an extra window to show more stats and game types than in the in-game one.

Session Stats

Show your session stats like in-game mods.

More capable

You can adjust the window so it can be suitable for any screen.

And more features

Frequently asked questions

We have some FAQ to
inform you more

Go to developer.hypixel.net to get the key.

Firstly, run /who command ingame. If still nothing happened, check you language. The Hypixel ingame language should be the same as the software language. Also check your log path.

If all these solutions don't work, report it to the developer.

There is a request limit in Mojang API. When reach the limit, the software will auto-retry to get data. This operation will take some time.

Also, your network environment is an important factor to cause this.
